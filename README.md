# VetWeb Project - "San Francisco de Asis" Veterinary Clinic

This is a professional website project developed for the "San Francisco de Asis" veterinary clinic (Family Business). The goal is to provide a platform where clients can learn about the services, browse products, and facilitate communication with the clinic.

# Technologies Used
* **HTML5:** Site structure.
* **CSS3:** Responsive design and custom styles.
* **JavaScript:** Dynamic functionality.
* **Git & GitHub:** Version control and deployment.

# Features
* **Home Page:** General information and featured services (**Work in Progress**).
* **Product Catalog:** Item visualization through a JSON file (**Work in Progress**).
* **Image Gallery:** Showcase of facilities and patients (**Work in Progress**).
* **Services Section:** Detailed information on medical care, pharmacy, and accessories (**Work in Progress**).

#  How to view the project locally
1. Clone the repository: 
   `git clone https://github.com/eduardospf14/Proyecto_VetWeb.git`
2. Open the `index.html` file in your favorite browser.
